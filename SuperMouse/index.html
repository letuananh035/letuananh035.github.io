Super Mouse
1.0
Tải nhanh
Thông tin:
+ Là phần mềm miễn phí.
+ Là phần mềm auto click.
+ Chế độ Only One(Đang hoàn thiện, nên chưa ổn định).
Hướng dẫn:
+ Tải về -> giải nén -> mở file .exe.
+ F1: để lấy vị trí.
+ F5: Chạy và dừng auto.
+ F2: Lấy thông tin tab dùng cho Only One.
